## English

### Supported Versions

We actively support and provide security updates for the following versions of the application:

| Version | Supported |
|---------|-----------|
| 1.0.x   | :white_check_mark: Yes |
| < 1.0.0 | :x: No |

### Reporting a Vulnerability

If you discover a security vulnerability, please **do not** open a public issue. Instead, report it privately.

You can report vulnerabilities using GitHub's **Private Vulnerability Reporting** feature via the Security tab of this repository.

Alternatively, you can contact the maintainer directly via email: `bashirta95@gmail.com`.

We will investigate all reports and aim to provide a fix or response within 48 hours.

## العربية

### الإصدارات المدعومة

نقوم حالياً بدعم وتقديم تحديثات الأمان للإصدارات التالية من التطبيق:

| الإصدار | مدعوم |
|---------|-----------|
| 1.0.x   | :white_check_mark: نعم |
| < 1.0.0 | :x: لا |

### الإبلاغ عن الثغرات الأمنية

إذا اكتشفت ثغرة أمنية، يرجى **عدم** فتح تذكرة (Issue) عامة. بدلاً من ذلك، قم بالإبلاغ عنها بشكل خاص.

يمكنك الإبلاغ عن الثغرات الأمنية باستخدام ميزة **الإبلاغ الخاص عن الثغرات (Private Vulnerability Reporting)** من خلال تبويب الأمان (Security) الخاص بهذا المستودع على GitHub.

بدلاً من ذلك، يمكنك الاتصال بالمطور مباشرة عبر البريد الإلكتروني: `bashirta95@gmail.com`.

سنقوم بالتحقق من جميع التقارير ونهدف إلى تقديم حل أو رد في غضون 48 ساعة.
